### drawing_tool.py

In this python file, you can control the pen as using some keys as w, s, d, a, r, g, b.

- w -> move forward.
- s -> move backward.
- a -> turn clockwise
- d -> turn counterclockwise

- r -> make pen red
- g -> make pen green
- b -> make pen blue

- c -> clear the canvas and go to start point.
